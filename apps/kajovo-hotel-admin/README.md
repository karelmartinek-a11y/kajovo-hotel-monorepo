# Kájovo Hotel Admin

Admin aplikace se samostatným loginem a správou provozních i administrativních modulů.

## Scope

- admin login
- dashboard a nastavení
- správa uživatelů a SMTP nastavení
- moduly `snídaně`, `ztráty a nálezy`, `závady`, `sklad`, `hlášení`
- utility stavy a role-based navigace

## Příkazy

```bash
pnpm dev
pnpm build
pnpm test:smoke
pnpm test:visual
```
