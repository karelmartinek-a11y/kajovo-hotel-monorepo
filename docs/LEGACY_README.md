# Legacy zdroje

Adresář `legacy/` obsahuje historické zdrojové materiály, šablony a assety.

Platí:

- nový vývoj patří do `apps/`, `packages/` a `android/`
- `legacy/` slouží jen pro referenci a forenzní dohledání starého stavu
- změny v `legacy/` dělat jen tehdy, když je to výslovně součást migračního nebo archivního úkolu
