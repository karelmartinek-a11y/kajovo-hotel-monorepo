# Archiv dokumentace

Tento adresář drží historické audity, jednorázové handoffy, forenzní poznámky a další materiály, které už nejsou current-state autoritou.

Platí:

- archiv slouží jako evidence,
- current-state rozhodování se řídí dokumenty uvedenými v `docs/README.md`,
- při konfliktu s kódem nebo workflow má přednost kód a guardy.
